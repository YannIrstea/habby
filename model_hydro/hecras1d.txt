Input: The .g0X file (geometry) and the xml file (results).

To obtain the xml file:
- Load the model in HEC-RAS (after it was run).
- Click on File, Export Geometry and Result (RAS Mapper)
- Export the needed variables and simulations.

This function works with steady and unsteady data.
